# AI 수학 튜터 (AI Math Tutor)
### 구현: [링크](https://delicate-bubblegum-7f4517.netlify.app/)
이 프로젝트는 Vue 3와 Google Gemini API를 사용하여 구축된 간단한 웹 애플리케이션입니다. 사용자가 원하는 유형과 난이도의 수학 문제를 AI에게 요청하고, 캔버스에 직접 손으로 쓴 풀이를 제출하면 AI가 이를 분석하여 상세한 피드백을 제공합니다.

## 주요 기능

맞춤형 문제 출제: 사용자가 원하는 '문제 유형' (예: 미적분, 이차함수)과 '난이도' (1~5단계)를 설정하여 AI로부터 맞춤형 수학 문제를 받을 수 있습니다.

디지털 필기 캔버스: 마우스나 터치(태블릿/모바일)를 지원하는 캔버스에 직접 풀이 과정을 필기할 수 있습니다.

도구: 검은색 펜, 빨간색 펜, 지우개, 전체 삭제 기능을 제공합니다.

영역 확장: '필기 영역 늘리기' 버튼으로 캔버스 높이를 동적으로 늘려 공간 부족 없이 풀이를 이어갈 수 있습니다.

AI 필기 분석 및 피드백: 캔버스에 작성된 필기 내용을 이미지로 변환하여 Gemini API (멀티모달)로 전송합니다. AI는 학생의 풀이 과정을 단계별로 분석하고, 채점 및 상세한 피드백을 제공합니다.

수학 수식 렌더링: MathJax 라이브러리를 통합하여, AI가 출제한 문제나 피드백에 포함된 LaTeX 형식의 수학 수식($f(x)$ 등)이 깨지지 않고 명확하게 표시됩니다.

## 사용 방법

API 키 준비:

Google AI Studio (또는 Google Cloud)에서 Gemini API 사용을 위한 API 키를 발급받습니다.

API 키 입력:

애플리케이션 상단의 'Google AI API Key' 입력란에 발급받은 키를 붙여넣습니다. (키는 브라우저에만 저장됩니다.)

문제 받기:

'문제 유형'을 입력하고 '난이도'를 선택합니다.

새로운 문제 받기 버튼을 클릭하면 AI가 조건에 맞는 문제를 출제합니다.

답안 작성:

'2. 답안 필기' 섹션의 캔버스에 풀이 과정을 작성합니다.

필요에 따라 펜 색상을 바꾸거나 지우개를 사용하고, 공간이 부족하면 필기 영역 늘리기 버튼을 누릅니다.

피드백 받기:

풀이 작성이 완료되면 피드백 받기 버튼을 클릭합니다.

AI가 캔버스의 필기 내용을 분석하는 동안 잠시 기다립니다.

'3. AI 피드백' 섹션에 AI 튜터의 상세한 분석과 채점 결과가 표시됩니다.

## 사용된 기술

Frontend: Vue 3 (CDN), Tailwind CSS (CDN)

AI Model: Google Gemini API (gemini-2.5-flash-preview-09-2025)

Rendering: MathJax (수학 수식 렌더링)

Core: HTML5 Canvas, JavaScript (ESM)
